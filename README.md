# Subscriber_vue

Console app that subscribe to a topic and print the mqtt msg on the screen

## Getting Started

Run inside the project folder:
```
npm i
```

to run the app type:
```
node .\server.js
```

### Prerequisites

Install [node](https://nodejs.org/it/download/)
